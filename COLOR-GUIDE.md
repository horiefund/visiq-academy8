# VISIQ Academy - カラーガイド（最新版）

## 🎨 公式カラーパレット

VISIQアイコンに基づく、5色のグラデーションカラーです。

### プライマリカラー

| カラー名 | カラーコード | Tailwind CSS | 用途 |
|---------|-------------|--------------|------|
| 🔵 **スカイブルー** | `#00A6E8` | `sky-400`, `sky-500` | メイン・リンク・アクセント |
| 🟢 **グリーン** | `#00C853` | `green-500` | サブカラー |
| 🟡 **イエロー** | `#FFD600` | `yellow-400`, `yellow-500` | アクセント・注目要素 |
| 🟠 **オレンジ** | `#FF6D00` | `orange-500` | CTA・強調 |
| 🔴 **レッド** | `#FF3838` | `red-500` | 最終アクセント |

---

## 🌈 グラデーション定義

### CSSグラデーション

```css
/* 135度のグラデーション（左上から右下） */
background: linear-gradient(135deg, 
    #00A6E8 0%,   /* スカイブルー */
    #00C853 25%,  /* グリーン */
    #FFD600 50%,  /* イエロー */
    #FF6D00 75%,  /* オレンジ */
    #FF3838 100%  /* レッド */
);

/* 90度のグラデーション（左から右） */
background: linear-gradient(90deg, 
    #00A6E8 0%,   /* スカイブルー */
    #00C853 25%,  /* グリーン */
    #FFD600 50%,  /* イエロー */
    #FF6D00 75%,  /* オレンジ */
    #FF3838 100%  /* レッド */
);
```

### Tailwind CSSクラス

```html
<!-- ボタングラデーション -->
<button class="bg-gradient-to-r from-sky-400 via-yellow-400 to-red-500">
    ボタン
</button>

<!-- テキストグラデーション -->
<span class="bg-gradient-to-r from-sky-400 via-yellow-400 to-red-500 bg-clip-text text-transparent">
    グラデーションテキスト
</span>
```

## 📋 カラー使用例

### テキストカラー


### ボーダーカラー
```html
<!-- スカイブルーボーダー -->

<input class="focus:border-sky-500" />
```

### ホバーエフェクト
```html
<!-- ホバー時にスカイブルー -->

```

## 🎯 使用場所

### ヒーローセクション
- 背景グラデーション: 135度の5色グラデーション
- 透明度: 5-10%（`opacity-5` または `opacity-10`）

### 見出し
- テキストグラデーション: 135度の5色グラデーション
- クラス: `.text-gradient`

### ボタン
- CTAボタン: `from-sky-400 via-yellow-400 to-red-500`
- ホバー時: シャドウとスケールアップ

### ロゴ
- 背景グラデーション: `from-sky-400 via-yellow-400 to-red-500`
- または: 実際のロゴ画像（logo.png）

### リンク
- デフォルト: `text-gray-700`
- ホバー時: `hover:text-sky-500`

### フォーム
- フォーカス時のボーダー: `focus:border-sky-500`

## ✨ アクセシビリティ

### コントラスト比

- スカイブルー（#00A6E8）+ 白背景: ✅ AAA
- イエロー（#FFD600）+ 白背景: ⚠️ 濃いテキストと組み合わせ推奨
- レッド（#FF3838）+ 白背景: ✅ AA

### 推奨事項

- イエローは背景色として使用し、テキストは濃い色を使用
- 重要な情報はスカイブルーまたはレッドを使用
- グラデーションは装飾的な要素に使用

---

## 🔄 カスタマイズ

グラデーションの方向や色の配分を変更したい場合：

```css
/* 上から下へ */
background: linear-gradient(180deg, #00A6E8 0%, #FF3838 100%);

/* 2色のシンプルグラデーション */
background: linear-gradient(135deg, #00A6E8 0%, #FF3838 100%);

/* 3色 */
background: linear-gradient(135deg, #00A6E8 0%, #FFD600 50%, #FF3838 100%);
```
